# Arquitectura del Sistema Project Brain

## Visión General
Project Brain es un sistema de inteligencia artificial especializado en comprensión, análisis y gestión evolutiva de proyectos de software. Actúa como un cerebro colectivo persistente que supera las limitaciones de contexto de los LLMs tradicionales.
